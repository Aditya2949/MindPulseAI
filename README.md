## 🎯 Problem Statement

Mental health issues like stress and anxiety are increasingly common among students and professionals, yet most people lack easy access to tools that help them **understand and monitor their emotional patterns** in real time.

MindPulseAI bridges this gap by providing an **AI-powered mental wellness companion** that is accessible, empathetic, and data-driven.

---

## 💡 What is MindPulseAI?

MindPulseAI is an intelligent mental health monitoring platform that uses **AI Reasoning Agents** to:

- Detect **stress, anxiety, and emotional patterns** from journal entries
- Provide a personalized **Wellness Score** (0–100)
- Offer **actionable insights and suggestions** tailored to the user
- Enable real-time **empathetic AI conversations** for mental support
- Track **daily check-ins** to visualize the wellness journey over time

---

## ✨ Features

| Feature | Description |
|---|---|
| 📋 **Daily Check-in** | Log mood, energy level, sleep hours, and journal entry |
| 🧠 **AI Wellness Analysis** | Detects emotion tags, stress & anxiety levels, generates wellness score |
| 💬 **AI Chat Companion** | Real-time supportive conversation with an empathetic AI agent |
| 📈 **Wellness Journey** | Track and visualize historical check-ins |
| ✨ **Daily Affirmations** | Personalized uplifting messages based on emotional state |

## 🛠️ Tech Stack

- **Frontend:** React.js (JSX), CSS-in-JS
- **AI Engine:** Claude claude-sonnet-4-20250514 (Anthropic) via REST API
- **Agent Type:** Reasoning Agent — multi-input emotional analysis
- **Deployment:** Runs in browser, no backend required

---

## 🚀 How It Works

```
User Input (Mood + Energy + Sleep + Journal)
        ↓
  Reasoning Agent (Claude AI)
        ↓
Emotional Pattern Analysis
        ↓
Wellness Score + Stress/Anxiety Levels + Insights
        ↓
  Personalized Suggestions + Affirmation
```

The AI agent takes **multi-modal inputs** (numerical ratings + free text) and performs:
1. **Sentiment Analysis** — understands emotional tone
2. **Stress Detection** — identifies stress indicators in language
3. **Anxiety Assessment** — detects anxiety patterns
4. **Wellness Scoring** — computes a 0–100 score
5. **Action Planning** — generates personalized, actionable suggestions

## 🏃 Getting Started

### Prerequisites
- Node.js 18+
- Anthropic API Key (free tier available)

### Installation

```bash
# Clone the repository
git clone https://github.com/Aditya2949/MindPulseAI.git
cd MindPulseAI

# Install dependencies
npm install

# Run the app
npm start
```

### Usage
1. Open the app in your browser
2. Go to **Daily Check-in** tab
3. Select your mood, energy level, sleep hours
4. Write how you're feeling in the journal
5. Click **Analyze My Wellness**
6. View your personalized wellness report!
